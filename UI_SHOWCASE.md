## 🎨 Design System

### Colors & Gradients
```
Primary: Indigo → Purple gradient
Success: Emerald → Teal gradient
Warning: Amber → Orange gradient
Error: Red → Rose gradient
Info: Blue → Cyan gradient
```

### Components
- **Cards**: Shadow depth, hover effects, smooth transitions
- **Buttons**: Gradient fills, shadow on hover, scale transforms
- **Tables**: Zebra striping, hover states, responsive
- **Stats**: Large numbers, trend indicators, progress bars
- **Empty States**: Encouraging messages, call-to-action
- **Loading States**: Shimmer effects, skeleton screens

### Animations
- **Fade In**: Smooth entry animations
- **Slide Up**: Cards animate from bottom
- **Scale**: Hover scale transforms
- **Shimmer**: Loading effect
- **All transitions**: 300ms ease-out

### Typography
- **Headings**: Bold, gradient text where appropriate
- **Body**: Clean, readable sans-serif
- **Numbers**: Large, bold for metrics
- **Descriptions**: Muted foreground color
