## 🏗️ TECHNICAL ARCHITECTURE

### Stack
- **Frontend**: Next.js 15.5.9 (App Router), React 18, TypeScript
- **Styling**: Tailwind CSS, Shadcn UI, Custom Design System
- **Backend**: Next.js Server Actions, API Routes
- **Database**: Supabase (PostgreSQL with RLS)
- **Auth**: Supabase Auth with RBAC
- **Payments**: Paystack, M-Pesa STK Push
- **Messaging**: Twilio WhatsApp API

### Security
- ✅ Row Level Security (RLS) on all tables
- ✅ Role-based access control (RBAC)
- ✅ Multi-tenant isolation
- ✅ Secure webhook verification
- ✅ JWT session tokens
- ✅ Environment variable protection

### Multi-Tenancy
- ✅ Shared database with `tenant_id`
- ✅ RLS policies for data isolation
- ✅ Hybrid deployment (shared/isolated)
- ✅ Tenant context management
